#comentarios

* Não irá afetar seu código
* Ajuda a lembrar blocos de códigos
* Deixa dicas para leitura
* Ajuda outros a entenderem
* Nunca esqueça de fechar um comentário aberto

Comentários com `/*` e termina com `*/`

```css

/* Básico */

/* ------------------------------------------ */
